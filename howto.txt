After creating repository on github.com:

In local directory:
  git init
  git add <filename>  [or git add -A]
  git commit -m "<comment>"
  git remote add origin https://github.com/bradplaster08/githubTest.git
  git push -u origin master

Pulling:
  git clone https://github.com/bradplaster08/githubTest.git
  git pull [thereafter]
